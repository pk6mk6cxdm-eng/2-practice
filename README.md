7)Аурухананы тіркегенде атауын беріңіз, пациент пен дәрігер санын кейін қосып, 
сыйымдылықтан асып кетуді тексеріңіз, дәрігер жетіспесе ескерту жасаңыз. 
7)Кітапхана, Кітап және Оқырман класстары бар. Кітап саны мен берілген кітаптар санын 
енгізіңіз. Кітап қайтарылған сайын жалпы санды жаңартып, кешіктірілген кітаптар көп болса 
ескерту көрсетіңіз, жаңа кітап қосылған сайын кітап санын есептеңіз.
